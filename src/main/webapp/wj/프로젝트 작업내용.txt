◈ 원준 작업

1. index.jsp

＊헤더 작업

＊스크롤 탑 (이미지 클릭시 맨위로) 작업

＊button 호버 (Fade,3D버튼 효과 적용)


2. home.jsp - 메인페이지

＊bxslider 작업 ( 이미지랑 규격 빠져나가지 않게 max-width:100% !importnat)

＊메인 컨텐츠 뼈대 디자인 확대, 축소 스크롤시 컨텐츠 css깨짐 방지를 위해 반응형으로 제작(max-width:100% 및 flex:wrap)


3. shop.jsp (쇼핑몰 페이지 메뉴 및 컨텐츠 뼈대 디자인)


4. itemDetail.jsp (쇼핑몰에서 상품 클릭시 나오는 상세페이지 제작 및 리뷰 별점 기능작업)


5. cart.jsp (장바구니 메뉴 및 컨텐츠 뼈대 디자인)


6. LoginPage2.jsp (로그인 화면 인풋및 전체 디자인)

＊사업자 로그인 버튼 클릭시 나오는 dialog 모달창 디자인 제작

＊id,pw 찾기 페이지 디자인 제작 결과 페이지 디자인


7.regUser,regSeller.jsp (회원가입 페이지 개인용,사업자용 디자인 제작)

＊비밀번호 변경 클릭시 나오는 dialog 모달창 디자인 제작

8. orderPage2.jsp (주문 제작 페이지 뼈대 디자인 및 js기능 구현)

＊detail태그 사용하여 <summary> 요소가 클릭되면 이벤트 핸들러가 실행

* 화살표 이미지 클릭시 화살표 각도 rotateZ(0deg) → rotateZ(90deg) 매끄럽게 바뀌는 기능

* 클릭시 컨텐츠 내용 나올때 ease-out으로 부드럽게 아코디언 효과적용
